# DSA - Luyện Cấu trúc dữ liệu & Giải thuật

## Mục đích

Repo học tập/luyện tập DSA theo lộ trình tự học, chuẩn bị phỏng vấn kỹ thuật và
rèn tư duy giải thuật. Có file `DSA_Roadmap.md` làm kim chỉ nam (roadmap chia
theo phase, checklist chủ đề), kèm code luyện tập từng dạng bài.

## Ngôn ngữ

- **Java** (các lời giải bài tập là file `.java`).
- Tài liệu lý thuyết viết bằng Markdown tiếng Việt.

## Chủ đề / dạng bài đã làm

- **Big O Notation** — lý thuyết + bài tính độ phức tạp (`P1_1_BigOCalculate`, `P1_2_Compare`).
- **Arrays** — MoveZero, RemoveDuplicateFromSortedArray, SortColors.
- **Two Pointers** — TwoSum, ThreeSum, ThreeSumClosest, ContainerWithMostWater, ValidPalindrome.
- **Sliding Window** — LongestSubstringWithoutRepeating, MinimumSizeSubArraySum, SubarraySumEqualsK.

Tài liệu theo chủ đề nằm ở `Phase 1 - Nền tảng/docs/` (vd `1_two_pointers.md`,
`2_sliding_window.md`); code bài tập ở `Phase 1 - Nền tảng/problems/`.

## Điểm đáng nhớ

- Cấu trúc phân theo **phase**: hiện đang ở Phase 1 (nền tảng: Big O, mảng, chuỗi, con trỏ, cửa sổ trượt).
- Mỗi dạng bài có cặp docs (lý thuyết) + problems (lời giải) song song — dễ ôn.
- Git còn sơ khai (vài commit `[wip]`), repo đang active, chưa hoàn thiện lộ trình.
